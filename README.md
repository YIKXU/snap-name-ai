Menu bar app to auto-rename your mac screenshots' goofy ahh timestamp names with local Ollama.

Features:
1. Pop-up for you to review the screenshot file name given by the AI, can tweak it directly from the popup.
 
2. Can choose which file the thingy has access to (make a 'screenshot' file on your desktop, so it doesnt have access to ur other files)

3. Add it to your login item if uw idk.

All these features can be turned on or off whenever via the menu bar icon.
