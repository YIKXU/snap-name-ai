No need to buy cleanshot x if you just want to auto rename your mac screenshots with local Ollama.
Features:
1. Pop-up for you to review the screenshot file name given by the AI, can tweak it directly from the popup, no need to open the whole file and edit it. 
2. Can choose which file the tool has access to (for example, if your screenshot saves on a 'screenshot' file on your desktop, can map it directly to that file only so it doesnt have access to other files)

All these features can be tweaked via the menu bar icon.
